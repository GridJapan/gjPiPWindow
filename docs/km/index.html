មុខងារ

PiP ច្រើនក្នុងពេលតែមួយ

អាចបើក PiP សម្រាប់អេក្រង់នីមួយៗបាន។ មួយៗជាការចាប់យកឯករាជ្យ។

ការកំណត់ដាច់ដោយឡែកសម្រាប់ PiP នីមួយៗ

កំណត់ការនៅលើគេជានិច្ច អត្រាស៊ុម គែមចេញ និងតម្រងផ្សេងគ្នាសម្រាប់បង្អួចនីមួយៗបាន។

នៅលើគេជានិច្ច

លំនាំដើមគឺ OFF។ បើបើក ON វានឹងមិនត្រូវបង្អួចផ្សេងៗបាំងទេ ប៉ុន្តែមិនបង្ហាញក្នុង Mission Control ឡើយ។

គ្រាន់តែរំកិលដល់គែមអេក្រង់ ក៏ត្រឡប់មកអេក្រង់ពិតវិញ

ការចេញពីរបៀបគ្រប់គ្រងគឺដូចការរំកិលចេញពីម៉ូនីទ័រ។ គែមចេញអាចកំណត់ ON/OFF សម្រាប់លើ ក្រោម ឆ្វេង ស្តាំបាន ហើយលំនាំដើមបិទតែគែមក្រោមប៉ុណ្ណោះ (ព្រោះជាទីតាំងបង្ហាញ Dock)។

អត្រាស៊ុម

អាចប្តូរ 30・60・120 fps សម្រាប់ PiP នីមួយៗបាន។

ចេញដោយបង្ខំ
